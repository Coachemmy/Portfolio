## Instructions


```shell
nvm install
nvm use
```

Or -

```shell
fnm install
fnm use
```

Install and run -

```shell
yarn
yarn start
```

### How to use

- Open `public/index.html` and replace -

`<title>John Smith</title>` with `<title>Your Name</title>`.

- Open `src/portfolio.js` and make changes.

- Build and deploy -

```shell
yarn build
yarn deploy
```
